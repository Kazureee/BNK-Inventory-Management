# Inventory Management System with Information Assurance & Security 🔐📦

## About the Project
This project is a web-based Inventory Management System designed to manage and monitor inventory records efficiently while ensuring that Information Assurance and Security principles are applied throughout the system.

The main objective of this project is to demonstrate how cybersecurity concepts can be integrated into a web application while maintaining proper inventory management.

## Key Features
- Inventory Tracking – Add, update, and monitor inventory items.
- User Authentication – Secure login system that allows only authorized users to access the system.
- Data Validation – Input validation to prevent incorrect or malicious data entries.
- Role-Based Access Control – Different user access levels to protect sensitive operations.
- Secure Data Handling – Basic security practices implemented to help protect stored information.

## Information Assurance and Security Implementation
The system follows several important Information Assurance principles:

- Confidentiality – Access to inventory data is restricted to authorized users only.
- Integrity – Ensures that inventory records remain accurate and protected from unauthorized modifications.
- Availability – Authorized users can reliably access the system and its data when needed.
- Access Control – User permissions are used to manage what actions users can perform within the system.

## Purpose of the Project
This project was developed as part of my portfolio to demonstrate my ability to design and build a secure web application while applying the fundamental concepts of Information Assurance and Security in a practical system.
